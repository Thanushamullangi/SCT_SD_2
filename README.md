Number Guessing Game (Python)
Project Description:

This is a simple Python-based number guessing game where the computer randomly selects a number between 1 and 10, and the user tries to guess it.

The program checks whether the user’s guess is correct and displays the result.

 Features:

Generates a random number between 1 and 10

Takes user input for guessing

Provides instant feedback (Correct/Wrong)

Displays the correct number if guessed wrong

Beginner-friendly project

How It Works:

The program generates a random number using Python’s random module.

The user enters a guess.

The program compares the guess with the generated number.

It displays the result.

Make sure Python is installed (Python 3.6+ recommended)
